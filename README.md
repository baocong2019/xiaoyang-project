*要求：*
*4.智能语音遥控风扇（要求可语音遥控+监测环境温度智能调速，论文需重点展开作品中的各通信协议，时序图要重点展开介绍，如iic协议，spi协议等，此外论文应大篇展开温度于风速之间的关系，使用pid算法调控）*

*基于pid算法控制下的智能语音遥控风扇
1.通过pid算法监测环境温度自动调速（温度与风速之间的关系）
2.客户端（小程序）可以远程控制
3.设定时间开启关闭（小程序/按键）
4.风扇的对面做个主控+温度检测+wifi，传输温度之类的信息
5.监测温度自动喷水 水里可以放半导体降温（待选）
6.语音控制模块asrpro*
